Internal Data Search Engine

A FastAPI + Next.js system to help teams discover datasets and search across data/metadata.

What it solves

  • Finding the right table quickly
  • Understanding column meanings (metadata-aware)
  • Reducing repeated internal “where is this data?” questions

Architecture

Next.js UI → FastAPI → structured datasets/metadata

Source

GitHub Repository

Run locally

pip install -r requirements.txt
uvicorn backend.main:app --reload
