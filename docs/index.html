Internal Data Search Engine

A system built to help teams search and understand internal datasets using keyword-based and metadata-aware search.

Problem

  • Hard to find the right table
  • Unclear column meanings
  • Repeated questions to data teams

Solution

A FastAPI backend with a Next.js UI that allows searching across datasets and schema metadata.

Architecture

Next.js UI
   ↓
FastAPI API
   ↓
Structured datasets / metadata
